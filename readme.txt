Для запуска:
1. Зайти в консоль
2. Перейти в директорию \VerbalMemoryTest-main\angular
3. прописать npm install
4. Зайти в новую консоль
5. Перейти в директорию \VerbalMemoryTest-main\api
6. Прописать php -S localhost:3000
7. Вернуться в прошлую консоль
8. В директории \VerbalMemoryTest-main\angular
   прописать npm start
в консоли с api будут логи сервера
в консоли angular будут логи с сайта
